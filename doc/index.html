Salza2 - Create compressed data from Common Lisp

Abstract

Salza2 is a library for creating compressed data in the ZLIB, DEFLATE, or GZIP data formats, described in RFC 1950, RFC 1951, and RFC 1952, respectively. It is written in pure Common Lisp and does not use any external libraries for compression. It does not yet support decompression.

Salza2 is available under a BSD-like license. The current version is 1.0, released on December XXXths, 2007.
